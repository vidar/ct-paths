How to use

  • Root — pick any content type (★ = has a url field)
  • Depth — expand/contract the whole tree to a fixed level
  • Click a node — toggle its children individually
  • Ignore — hide up to 5 content types from the tree
  • Top-Down / Left-Right — switch orientation
  • Reset View — re-center and fit to screen
  • Scroll to zoom, drag to pan
  • Dashed red nodes are cycles — already visited on this path
